# Code of Conduct

NewsFlow 希望保持一个清楚、尊重、能认真讨论问题的协作环境。

## 我们鼓励

- 基于事实和上下文讨论问题
- 尊重不同经验背景和技术判断
- 对系统边界、风险和取舍保持诚实
- 把批评放在方案和实现上，而不是针对个人

## 我们不接受

- 人身攻击
- 歧视、羞辱或威胁性表达
- 故意发布敏感信息或他人隐私
- 明知会伤害社区协作氛围的挑衅行为

## 处理方式

维护者可以对不合适的内容进行提醒、隐藏、关闭讨论或限制参与。

## 原则

NewsFlow 关注的是高责任场景中的系统设计与人机协同。我们希望社区讨论也保持同样的态度：认真、克制、清楚、可追溯。
