# GitHub About Copy for NewsFlow

## Short description

An AI-assisted newsroom workflow system for event tracking, evidence organization, approval chains, correction handling, and post-publication monitoring.

## Alternative short description

A human-in-the-loop newsroom workflow system for events, evidence, drafting, approval, corrections, and post-publish follow-up.

## One-paragraph introduction

NewsFlow is an open-source newsroom workflow prototype built around events, Story Packets, evidence, sign-off, and correction loops. Instead of treating AI as a pure writing engine, it treats AI as a support layer for organizing the middle of the editorial process: lead intake, evidence structuring, risk exposure, approval traceability, and post-publication follow-up.
